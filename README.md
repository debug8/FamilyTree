# Family Tree

Настільний застосунок для Windows (**C# / WPF, .NET 10**) для збереження, редагування та візуалізації генеалогічного дерева
з автоматичним визначенням ступеня родинного зв'язку між будь-якими двома особами.

Користувач веде базу осіб родини (люди, зв'язки «батько/мати → дитина» та «подружжя»),
а застосунок будує дерево предків або нащадків, показує всіх родичів обраної особи й підписує назву
родинного зв'язку кожного відносно кореневої особи (наприклад: «тітка», «двоюрідний брат», «прабаба»). 
Інтерфейс мультимовний (українська / англійська), включно з логікою назв родства для кожної мови.

## Технології

- .NET 10 (LTS), C#, WPF
- Архітектура MVVM (`CommunityToolkit.Mvvm`)
- DI через `Microsoft.Extensions.Hosting` (Generic Host)
- Зберігання: файл-документ `.familytree` (JSON), без СУБД
- Тести: xUnit + Shouldly (моки — NSubstitute)
